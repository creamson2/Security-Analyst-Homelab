## 00 – Homelab Infrastructure
This folder documents the core setup of my Security Analyst Homelab.
It includes the base systems, virtualization, networking layout, and the environment where all other projects run.

## What’s included
- Virtualization setup (VirtualBox)

- Host machine configuration

- VM resources (CPU, RAM, storage)

- Network layout for the lab

- How each system connects (Kali, Wazuh, Splunk, Elasticsearch)

- Notes on stability and performance

## Purpose
This folder shows the foundation of my lab — the environment where all tools, tests, and investigations take place.
It helps me understand how everything is connected and ensures the lab stays stable as I build more projects.
