## 05 – VeraCrypt Encrypted Volume
This folder documents how I created and used a VeraCrypt encrypted volume inside my Security Analyst Lab.
It includes the setup steps, encryption settings, mounting process, and how I used it to securely store sensitive lab data.

## What’s included

- Creating a VeraCrypt encrypted container

- Encryption algorithm and volume settings

- Mounting and unmounting the volume

- Using the encrypted space for logs, tools, or notes

- Troubleshooting mounting issues

- Why encryption matters in a security lab

## Purpose
This folder shows how I apply encryption to protect sensitive files inside my lab.
It demonstrates practical use of secure storage — a key skill for handling logs, evidence, and investigation data safely.
