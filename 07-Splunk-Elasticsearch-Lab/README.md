## 07 – Splunk & Elasticsearch Lab
This folder documents my experiments, tests, and troubleshooting with Splunk and Elasticsearch inside my Security Analyst Lab.
It includes installation attempts, configuration notes, indexing behavior, and lessons learned — even when Elasticsearch did not fully work.

## What’s included
- Splunk installation and setup

- Testing log ingestion in Splunk

- Elasticsearch installation attempts

- Indexer and dashboard issues

- Configuration notes and troubleshooting steps

- What worked, what failed, and what I learned

## Purpose
This folder shows my hands-on experience with two major log analysis platforms.
Even when Elasticsearch didn’t run successfully, the testing and troubleshooting process is valuable — it demonstrates real Security Analyst problem‑solving and lab experimentation.
