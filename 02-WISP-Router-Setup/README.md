## 02 – WISP Router Setup

This folder documents how I configured a WISP router to support my isolated homelab network.
It includes the setup steps, network behavior, and how the router interacts with VirtualBox and my lab systems.

## What’s included

- WISP router configuration

- DHCP, NAT, and routing behavior

- How the router connects to the lab network

- Why I used a WISP router for isolation

- Notes on connectivity issues and fixes

## Purpose

This setup helps keep my homelab traffic separate from my main home network.
It also provides a stable, controlled environment for testing tools like Wazuh, Splunk, and Kali without affecting other devices.
