## 04 – Kali Linux Stability
This folder documents the issues and fixes related to keeping Kali Linux stable inside my Security Analyst Lab.
It includes networking problems, update failures, VirtualBox issues, and performance adjustments.

## What’s included
- Network connectivity problems

- Update and upgrade errors

- VirtualBox Guest Additions issues

- Display, resolution, and clipboard problems

- Slow performance and resource tuning

- Fixes that improved reliability

## Purpose
This folder shows how I troubleshoot and stabilize a core security tool.
Kali is essential for testing, scanning, and investigations, so keeping it reliable is a key part of maintaining a functional Security Analyst Lab.
