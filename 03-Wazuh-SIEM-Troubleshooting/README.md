## 03 – Wazuh SIEM Troubleshooting

This folder documents the issues I encountered while setting up and running Wazuh in my Security Analyst Lab.
It includes errors, fixes, configuration notes, and lessons learned while stabilizing the SIEM.

## What’s included

- Wazuh manager and agent connectivity issues

- Indexer and dashboard problems

- Service failures and restarts

- Log ingestion problems

- Network communication troubleshooting

- Configuration adjustments that improved stability

## Purpose

This folder shows how I troubleshoot real SIEM issues.
It demonstrates my ability to diagnose problems, test solutions, and document fixes — core skills for a Security Analyst working with Wazuh or any SIEM platform.
