## 08 – Security Analyst Workflow
This folder documents the core workflow I follow as a Security Analyst inside my lab environment.
It shows how I approach alerts, logs, investigations, and decision‑making in a structured, repeatable way.

## What’s included
- How I review alerts

- How I analyze logs from different sources

- How I triage events and prioritize work

- Investigation steps and decision trees

- Documentation habits and evidence collection

- Lessons learned from real lab scenarios

## Purpose
This folder demonstrates my understanding of Security Analyst responsibilities.
It shows that I can follow a consistent workflow, investigate events, document findings, and make informed decisions — all essential skills for real-world security operations.
