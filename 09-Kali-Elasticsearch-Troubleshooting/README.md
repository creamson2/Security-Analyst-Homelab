## 09 – Kali & Elasticsearch Troubleshooting
This folder documents the issues I encountered while working with Kali Linux and Elasticsearch inside my Security Analyst Lab.
It includes installation failures, service errors, networking problems, and the steps I took to diagnose and understand each issue.

## What’s included
- Elasticsearch installation attempts

- Service startup failures and error messages

- Network and port accessibility issues
 
- Kali Linux connectivity problems

- Resource limitations and performance issues

- What worked, what failed, and what I learned

## Purpose
This folder shows my ability to troubleshoot complex systems, even when the outcome isn’t perfect.
Real Security Analysts often deal with broken services, failed installs, and unexpected behavior — documenting the process is part of the job.
This folder demonstrates persistence, problem‑solving, and technical curiosity.
